## Usage

Put these lines in your `Cargo.toml`, under `[dependencies]`.

```
inquire = "0.7.1"
inquire-derive = "0.7.1"
```

```
inquire = { version = "0.7.1", features = ["date", "editor"] }
inquire-derive = { version = "0.7.1" }
```
